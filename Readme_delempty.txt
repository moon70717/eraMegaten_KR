0KB이거나 주석만 있는 빈 파일을 삭제하는 스크립트

사용법
delempty.bat과 delempty.ps1을 eraMegaten 폴더에 복사한 후 delempty.bat 실행.
실행이 끝나면 삭제 로그가 delempty.log로 출력된다.

PowerShell 스크립트이기 때문에 다른 것은 설치할 필요가 없을 것이다. 에러가 날 것 같으면 Windows Update

파일을 삭제하는 것이므로 걱정되는 경우는 실행 전에 백업을 해 두는 등, 자기 책임 하에 사용할 것

스크립트는 라이선스가 없습니다.


=========================================================================================================


0KBだったりコメントだけの空ファイルを削除するスクリプト

使い方
delempty.batとdelempty.ps1をeraMegatenフォルダにコピーしてdelempty.batを実行。
実行が終わったら削除ログがdelempty.logに出力される。

PowerShellスクリプトなので他に何かインストールする必要はないはず。エラーが出るようならWindowsUpdate

ファイルを削除するものなので、心配な場合は実行前にバックアップを取っておくなどして、自己責任の上で使用してください。

スクリプトはライセンスフリーです。
